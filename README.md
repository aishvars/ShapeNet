# ShapeNet Part Segmentation — 24-788 Deep Learning Project

We compare three point cloud architectures on the ShapeNet part segmentation benchmark:
**PointNet** (baseline), **PointNet++**, and **DGCNN**, evaluated on 16 object categories
and 50 part labels across 16,880 3D shapes.

## Repository Structure

- `IDL_ShapeNet_project_final_code.ipynb` — full training pipeline (data loading, model definitions, training, evaluation, figures)
- `ShapeNet_reproduce_code.ipynb` — loads trained checkpoints and regenerates all reported metrics and figures without retraining

## Reproducing Results

Open `ShapeNet_reproduce_code.ipynb` on Kaggle and follow the setup instructions at the top of the notebook. It covers:
- Attaching the trained checkpoints dataset
- Setting up your Kaggle API token for the ShapeNet download
- Expected runtime and output locations

All figures and metrics from the report are regenerated by running that notebook end-to-end.

## Dependencies

- PyTorch
- PyTorch Geometric
- NumPy, Matplotlib
